MAPeado para todo publico

Permite crear un nuevo arreglo aplicando una función a cada elemento del arreglo original, sin modificarlo.

FILTER

Genera un nuevo arreglo con los elementos que cumplen una condición específica establecida en una función.

REDUCE

Combina todos los elementos de un arreglo en un solo valor, aplicando una función acumuladora paso a paso.

FIND

Devuelve el primer elemento del arreglo que cumple una condición determinada; si no existe, retorna undefined.

SORT

Ordena los elementos de un arreglo según un criterio definido, ya sea de forma alfabética o numérica.

INCLUDES

Verifica si un arreglo contiene un elemento específico y devuelve true o false según el resultado.

INDEX OF

Indica la posición del primer elemento que coincide con el valor buscado dentro del arreglo, o -1 si no se encuentra.

SOME

Comprueba si al menos un elemento del arreglo cumple una condición dada; devuelve true o false.

EVERY

Evalúa si todos los elementos del arreglo cumplen una condición específica; devuelve true solo si todos lo hacen.

SLICE

Copia una parte del arreglo original y devuelve un nuevo arreglo sin modificar el original.
